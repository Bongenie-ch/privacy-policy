Data Deletion Instructions

Requesting deletion of your data

This website uses the Instagram Graph API provided by Meta Platforms to display Instagram content.

We do not store personal user data. However, if you would like to request deletion of any data associated with you, you can contact us at:

Email: opertin@bongenie.ch

What happens when you request deletion

If any data related to your request exists (such as cached API data), it will be permanently removed as soon as possible.

If no data is stored, we will confirm that no personal data is retained.
